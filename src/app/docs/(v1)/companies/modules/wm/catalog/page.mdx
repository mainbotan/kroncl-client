# 1. Каталог

<span className={styles.underline}>«Каталог»</span> — подсистема, определяющая товарное предложение компании (услуги и товары). Каталог состоит из <span className={styles.underline}>«Категорий»</span> и <span className={styles.underline}>«Товарных позиций»</span>, которые в совокупности определяют каталог организации.

Таким образом, каталог — это шаблон, определяющий, чем и в каком количестве торгует ваша организация. Категории помогают структурировать каталог, а товарные позиции описывают каждый элемент каталога (услугу или товар).

Модуль <Link href='/docs/companies/modules/dm' className={styles.link}>«Сделки»</Link> тесно интегрирован с каталогом, позволяя указывать услуги, товары или конкретные позиции из поставок в качестве состава заказа.

# 2. Где это?

Управление категориями и товарными позициями каталога доступно на соответствующих вкладках раздела <span className={styles.underline}>«Каталог»</span> (левая боковая панель пространства организации).
