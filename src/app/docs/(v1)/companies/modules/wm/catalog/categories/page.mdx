# 1. Категории

<span className={styles.underline}>Категории</span> используются для разделения каталога на логические области, отражающие структуру самой компании. Большинство компаний, вне зависимости от размера и специфики, используют разделение каталога на разные предметные области.

Таким образом, компания «А», предоставляющая товары и услуги в сфере автообслуживания, может использовать разделение каталога на категории:

- МКПП (механические коробки переключения передач)
- АКПП (автоматические коробки переключения передач)

<br />

Каждая из таких категорий содержит свои услуги и товарные позиции, кроме того, может иметь подкатегории, например:

- штоки
- вилки

<br />

Аналогичное разделение каталога применимо к подавляющему количеству отраслей производства и обслуживания.

# 2. Просмотр категорий

Просмотр существующих категорий доступен на вкладке <span className={styles.underline}>«Категории»</span> раздела <span className={styles.underline}>«Каталог»</span>.

<MDXImageBlock src='/images/docs/company-wm-catalog-categories.png' label='Просмотр категорий каталога.' />

# 3. Создание категории

Для создания категории нажмите на соответствующую кнопку на странице просмотра всех категорий.

Каждую категорию описывает ряд обязательных и опциональных полей:

- наименование
- комментарий (опционально)
- родительская категория (опционально)

<MDXImageBlock src='/images/docs/company-wm-catalog-create-category.png' label='Создание категории.' />

# 4. Просмотр категории

Для просмотра конкретной категории наведите курсор на карточку категории (страница просмотра коллекции) и нажмите на соответствующую кнопку.

На странице категории будут доступны действия редактирования и деактивации/активации.

# 5. Деактивация категории

Физическое удаление записи категории невозможно. Смена статуса активности категорий реализуется с помощью активации/деактивации, которые не удаляют запись, а скрывают категорию для последующих действий.
