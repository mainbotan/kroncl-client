# 1. Товарные позиции

<span className={styles.underline}>Товарные позиции</span> — элементы каталога, описывающие услуги и товары товарного предложения организации.

Важно: товарные позиции <span className={styles.underline}>не описывают фактическое существование товаров</span> в данный момент на складе. Поэтому вы не увидите привычных для бывших пользователей Битрикс/1С полей остатков в карте товарной позиции. Остатки на складе рассчитываются с помощью <Link href='/docs/companies/modules/wm/movement-products/batches' className={styles.link}>«Поставок и Отгрузок»</Link>.

Таким образом, товарная позиция описывает цены услуг и товаров, но не фиксирует доступность товаров в данный момент для продаж.

# 2. Просмотр товарных позиций

Просмотр товарных позиций доступен на соответствующей вкладке раздела <span className={styles.underline}>«Каталог»</span> (левая боковая панель пространства организации).

<MDXImageBlock src='/images/docs/company-wm-catalog-positions.png' label='Товарные позиции.' />

# 3. Создание товарной позиции

Создание товарной позиции доступно при нажатии на соответствующую кнопку на странице просмотра товарных позиций.

В зависимости от типа товарной позиции — <span className={styles.underline}>услуга</span> или <span className={styles.underline}>товар</span> — при создании требуются разные наборы обязательных и опциональных полей.

Общие поля для всех товарных позиций:

- категория (обязательна для создания)
- наименование (обязательное поле)
- тип (услуга/товар)
- комментарий (опционально)

<br />

# 3.1 Услуга

Для создания услуги доступен следующий набор полей:

- единица измерения
- базовая цена — цена оказания услуги, может быть изменена в конкретных заказах.

<br />

# 3.2 Товар

Kroncl позволяет адаптировать каталог как под материальные товары, так и под разные формы цифровых продаж (например, лицензии).

Для всех товаров доступен следующий набор полей:

- тип учёта — складской учёт / без учёта (для цифровых товаров);
- детализация учёта — партионный / поштучный;
- метод учёта партий — FIFO / LIFO (первый пришёл — первый продан / последний пришёл — первый продан);
- единица измерения (штука, литр, кг, г, метр...).

# 4. Просмотр товарной позиции

Просмотр товарной позиции доступен сразу после создания или при переходе на страницу позиции при нажатии на её карточку.

Обращаем внимание читателя на идентификаторы товарных позиций (уникальные 8-символьные значения), которые могут использоваться для маркировки товаров.

<MDXImageBlock src='/images/docs/company-wm-catalog-positions-ids.png' label='Идентификаторы позиций.' />

На странице товарной позиции доступны действия редактирования и деактивации.

<MDXImageBlock src='/images/docs/company-wm-catalog-position.png' label='Товарная позиция.' />

# 5. Деактивация позиции

Kroncl запрещает физическое удаление записи товарной позиции. Регуляция статуса активности обеспечивается с помощью деактивации, которая не удаляет запись из связанных объектов (например, заказов), а скрывает товарную позицию для дальнейшего использования.

<MDXImageBlock src='/images/docs/company-wm-catalog-deactivate-position.png' label='Деактивация товарной позиции.' />
