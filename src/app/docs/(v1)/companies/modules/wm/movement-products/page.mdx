# 1. Движение товаров

<span className={styles.underline}>«Движение товаров»</span> на платформе Kroncl — история поставок и отгрузок на складе организации. 
<span className={styles.underline}>Каталог</span> описывает товарное предложение компании (что, по какой цене закупается/продаётся), а поставки и отгрузки показывают
фактическое наличие материальных товаров в ассортименте.

В рамках Kroncl и <span className={styles.underline}>поставки</span>, и <span className={styles.underline}>отгрузки</span> являются одним типом документа, различие между 
которыми заключается лишь в направлении движения товаров.

# 2. Просмотр поставок и отгрузок

Для просмотра существующих документов движения перейдите в раздел <span className={styles.underline}>«Каталог»</span> (левая боковая панель пространства организации) и откройте вкладку
<span className={styles.underline}>«Движение товаров»</span>.

<MDXImageBlock src='/images/docs/company-wm-batches.png' label='Просмотр поставок/отгрузок.' />

Карточка каждого документа содержит базовые поля:

- тип (поставка или отгрузка)
- идентификатор (8-символьный код)
- дата создания
<br />
Для перехода на страницу документа нажмите на соответствующую карточку.

# 3. Создание документа

Для создания документа (поставки или отгрузки) нажмите на соответствующее действие на странице просмотра всех уже созданных документов.

<MDXImageBlock src='/images/docs/company-wm-create-batch-search.png' label='Поиск товарных позиций в составе документа.' />

В теле документа находится поиск по <Link href='/docs/companies/modules/wm/catalog' className={styles.link}>каталогу</Link> организации и перечень
выбранных товаров. При вводе идентификатора/названия товарной позиции из каталога вам будут предложены наиболее подходящие варианты из всех уже зарегистрированных.
Чтобы добавить товарную позицию в состав поставки или отгрузки, достаточно нажать на соответствующую карточку в модальном списке.

Важно: из поисковой выборки Kroncl автоматически исключает товарные позиции с типом учёта = <span className={styles.underline}>без учёта</span>, включая все услуги. 
<Link href='/docs/companies/modules/wm/catalog/products' className={styles.link}>Подробнее о типах учёта товарных позиций</Link>.

<MDXImageBlock src='/images/docs/company-wm-create-income-batch.png' label='Создание документа.' />

# 3.1 Состав документа

Каждая выбранная товарная позиция в каждой конкретной поставке содержит базовую цену (продажи или закупки <span className={styles.underline}>в зависимости от типа документа</span>), количество
и идентификатор соответствующей товарной позиции.

Акцентируем внимание читателя на том, что <span className={styles.accent}>базовая цена единицы</span> товарной позиции зависит от направления документа: в случае поставки поле будет заполнено ценой закупки товарной позиции, в случае отгрузки — ценой продажи. 
При этом цены товаров можно переопределять для каждой конкретной поставки/отгрузки.

Для уменьшения или увеличения количества единиц товара нажмите на соответствующее действие в правой части карточки товара, для перехода на страницу соответствующей товарной позиции — на идентификатор (выделяется акцентным цветом).

# 4. Просмотр документа

После создания документа поставки/отгрузки станет доступна страница просмотра.

<MDXImageBlock src='/images/docs/company-wm-income-batch.png' label='Просмотр документа.' />

Редактировать уже созданный документ, включая его состав, невозможно. Удаление также запрещено в целях консистентности данных.
