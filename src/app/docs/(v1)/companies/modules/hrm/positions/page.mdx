# 1. Должности

Должности обеспечивают иерархию в штате сотрудников, позволяя давать определённым группам (например, руководству) большее
количество разрешений в рамках организации.

# 2. Просмотр должностей

Просмотр созданных должностей доступен в разделе <span className={styles.underline}>«Сотрудники»</span> (левая боковая панель пространства организации) → секция 
<span className={styles.underline}>«Должности»</span> в шапке страницы.

Для перехода на страницу конкретной должности достаточно навести курсор на соответствующую карту.

<MDXImageBlock src='/images/docs/company-hrm-positions.png' label='Просмотр должностей.' />

# 3. Создание должности

Создание должности доступно при нажатии на кнопку <span className={styles.underline}>«Создать»</span> на странице просмотра должностей.

<MDXImageBlock src='/images/docs/company-hrm-position-create.png' label='Создание должности.' />

Некоторые поля карты должности:

- наименование
- описание (опционально)
<br />

После создания произойдёт перенаправление на страницу должности.

# 4. Просмотр должности

На странице должности по аналогии с картой сотрудника доступны действия удаления, редактирования данных должности и 
привязка разрешений.

<MDXImageBlock src='/images/docs/company-hrm-position.png' label='Просмотр должности.' />

# 5. Разрешения

Должность может расширять разрешения всем назначенным на неё сотрудникам.
Работа с разрешениями должности доступна на странице просмотра карты должности.

Для привязки нового разрешения начните вводить латинский код в поисковик разрешений — система покажет вам наиболее подходящие 
варианты с расшифровкой значений каждого разрешения.

Список всех разрешений Kroncl доступен в главе <Link href='/docs/companies/accesses/permissions' className={styles.link}>«Разрешения»</Link>.

Для удаления разрешения из списка уже назначенных нажмите на крестик рядом с латинским кодом нужного разрешения.

<MDXImageBlock src='/images/docs/company-hrm-position-permissions.png' label='Разрешения должности.' />

# 6. Удаление должности

В отличие от карты сотрудника, должность можно безвозвратно удалить. Для этого нажмите на кнопку «Удалить» на странице просмотра 
должности, после чего подтвердите действие.

Удаление должности не повлечёт за собой удаление всех связанных сотрудников, однако приведёт к сбросу разрешений аккаунтов, привязанных к картам
этих сотрудников.

<MDXImageBlock src='/images/docs/company-hrm-position-delete.png' label='Удаление должности.' />
