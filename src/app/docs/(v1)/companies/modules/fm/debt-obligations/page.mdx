# 1. «Долговые обязательства»

«Долговые обязательства» - вторая по значению подсистема модуля <Link href='/docs/companies/modules/fm' className={styles.link}>«Финансы»</Link> 
после <Link href='/docs/companies/modules/fm/movement-of-funds' className={styles.link}>«Движения средств»</Link>.

Эта подсистема выполняет две основные функции:

- ведение базы контрагентов кредиторов/дебиторов
- учет дебетов/кредитов
<br />

Kroncl не пытается стать сложным инструментом, специализирующимся на расчетах специфических процентных ставок
долговых обязательств, и уж тем более не пытается заменить специалиста по просчету рисков. Вместо этого Kroncl 
дает вам место, где все долговые обязательства элементарным образом будут учтены и структурированы, чтобы ничто и никто не был забыт.

# 2. Где это?

Управление долговыми обязательствами доступно из раздела <span className={styles.underline}>«Финансы»</span> (левая боковая панель пространства организации).
