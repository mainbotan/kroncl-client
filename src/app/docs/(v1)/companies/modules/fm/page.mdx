# 1. «Финансы»

Модуль «Финансы» предоставляет функционал для работы с двумя основообразующими 
финансовыми подсистемами:

- <Link href='/docs/companies/modules/fm/movement-of-funds' className={styles.link}>«Движение средств»</Link>
- <Link href='/docs/companies/modules/fm/debt-obligations' className={styles.link}>«Долговые обязательства»</Link>
<br />

Управление финансами, включая две эти подсистемы, доступно в разделе <span className={styles.underline}>«Финансы»</span> (левая боковая панель пространства организации).

# 2. До прочтения модуля

В главах этого модуля используются перекрестные ссылки и термины из следующих разделов документации:

- <Link href='/docs/companies/modules/hrm/employees' className={styles.link}>Сотрудники</Link>: <span className={styles.secondary}>записи сотрудников используются при назначении ответственных за финансовые операции.</span>
<br />
Рекомендуем начать чтение этого модуля после прочтения указанных глав.
