# 1. Баланс предприятия

Баланс предприятия характеризует состояние финансов компании в данный момент. Эта глава не будет большой, 
но от этого не потеряет своей важности.

Основная идея Kroncl заключается в том, что <span className={styles.accent}>весь учет компании нужен в первую очередь для получения операционных показателей</span>. 
Такой же принцип применим для модуля <span className={styles.underline}>«Финансы»</span>. Все учтенные менеджерами/сотрудниками операции призваны в первую очередь для получения 
одного единственного числа - <span className={styles.underline}>баланса предприятия</span>.

# 2. Где это?

Баланс предприятия доступен на главной странице раздела <span className={styles.underline}>«Финансы»</span> (левая боковая панель пространства организации).

<MDXImageBlock src='/images/docs/company-balance.png' label='Баланс предприятия.' />

Здесь же доступны другие важные финансовые показатели, включая:

- количество операций за все время
- средний чек операции

<Link className={styles.link} href='/docs/companies/modules/fm/movement-of-funds/analysis'>«Анализ операций»</Link>.
