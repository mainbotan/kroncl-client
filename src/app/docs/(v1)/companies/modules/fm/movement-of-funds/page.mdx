# 1. «Движение средств»

Подсистема «Движение средств» модуля «Финансы» позволяет анализировать операции организации за целевой промежуток времени,
распределять траты и доходы по категориям и вести полную финансовую историю компании.

Центральные сущности подсистемы:

- <Link href='/docs/companies/modules/fm/movement-of-funds/categories' className={styles.link}>категория</Link>
- <Link href='/docs/companies/modules/fm/movement-of-funds/operations' className={styles.link}>операция</Link>
<br />

Одной из особенностей реализации финансовой истории Kroncl является отказ от ведения базы контрагентов для финансовых операций.
Это значит, что каждая операция может не ссылаться на получателя/отправителя средств, минимизируя действия менеджера.

Такой подход выбран нами специально, по причине того, что для предприятий малого бизнеса гораздо большее значение имеет объем операций и их
общая классификация на категории и тип (доход/расход).

# 2. Где это?

При открытии раздела «Финансы» (левая боковая панель пространства организации) вы увидите главное меню управления операциями. Здесь же
находится история транзакций (<span className={styles.secondary}>операций</span>), баланс предприятия и разделы навигации: категории, аналитика, долговые обязательства.

# 3. Использование в модулях

Подсистема «Движение средств» многократно переиспользуется в других модулях Kroncl. Таким образом, модуль <Link className={styles.link} href='/docs/companies/modules/dm'>«Управление сделками»</Link>
предоставляет действие «Расчет сделки», анализирующее состав заказа и автоматически создающее финансовую операцию дохода в модуле финансов.

Аналогичный функционал предоставляется в других модулях, включая 
<Link href='/docs/companies/modules/wm' className={styles.link}>«Каталог & Склад»</Link> (<span className={styles.secondary}>создание поставки/отгрузки создает соответствующую финансовую операцию</span>).
