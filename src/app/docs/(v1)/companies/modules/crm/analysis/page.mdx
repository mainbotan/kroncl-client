# 1. Анализ клиентской базы

Платформа Kroncl предоставляет методы анализа клиентского трафика организации.
Для просмотра аналитики перейдите в раздел <span className={styles.underline}>«Клиенты»</span> (боковая панель пространства организации), затем откройте вкладку <span className={styles.underline}>«Анализ»</span>. 

<MDXImageBlock src='/images/docs/company-crm-analysis-summary.png' label='Анализ клиентской базы.' />

# 2. Период анализа

По умолчанию система показывает аналитику за <span className={styles.underline}>весь период</span> существования организации. Для изменения
периода введите целевые начальную и конечную даты, после чего нажмите на кнопку, применяющую фильтры.

<MDXImageBlock src='/images/docs/company-crm-analysis-period.png' label='Период анализа.' />

# 3. Суммарная статистика

Суммарная статистика представлена основными показателями и графиком динамики изменения клиентской базы.

В числе основных показателей:

- общее количество клиентов
- новые за выбранный период
- активные
- неактивные
- физические лица
- юридические лица
<br />
График динамики визуализирует рост/снижение количества клиентов в базе организации.

# 4. Сравнение по ресурсам

Для просмотра сравнения показателей клиентской базы с категоризацией по <Link href='/docs/companies/modules/crm/sources' className={styles.link}>ресурсам привлечения</Link> перейдите на вкладку 
<span className={styles.underline}>«По источникам»</span>. Каждый ресурс привлечения представлен столбцом, соответствующим количеству созданных клиентов.
