# 1. Статусы сделок

<span className={styles.underline}>Статус сделки</span> определяет прогресс её обработки. По умолчанию Kroncl предоставляет
ряд статусов, которые вы можете переопределить в любой момент использования платформы.

Статусы используются в главном канбане сделок, выделяются цветами и информируют менеджера о количестве сделок в соответствующей стадии.

<MDXImageBlock src='/images/docs/company-dm-kanban-statuses.png' label='Статусы сделок в канбане.' />

# 2. Просмотр статусов

Для просмотра всех статусов перейдите на вкладку <span className={styles.underline}>«Статусы»</span>. На странице
будут представлены все существующие статусы, расположенные в порядке: от самой ранней стадии до последней.

Для изменения порядка статусов зажмите соответствующую карточку и переместите в нужное место. 
Пометка <span className={styles.accent}>«Начальный статус»</span> указывает на статус, в который попадают все новые сделки сразу после создания. 
При этом такой статус не обязательно должен располагаться первым.

Для редактирования статуса нажмите на соответствующую кнопку в карточке статуса.

<MDXImageBlock src='/images/docs/company-dm-statuses.png' label='Просмотр статусов.' />

# 3. Создание статуса

Для создания нового статуса нажмите на соответствующую кнопку на странице просмотра всех записей. 

Для инициализации нового статуса требуется ряд обязательных и опциональных полей:

- наименование
- цвет
- тип (обычный / начальный статус новых сделок, <span className={styles.secondary}>если был до этого для другого статуса — будет переопределён</span>)
- комментарий (опционально)

<MDXImageBlock src='/images/docs/company-dm-create-status.png' label='Создание статуса.' />

# 4. Просмотр статуса

Для просмотра статуса нажмите на кнопку редактирования в карточке статуса. На странице просмотра будут доступны действия над статусом,
включая <span className={styles.accent}>редактирование</span> и <span className={styles.accent}>удаление</span>.

Вы можете переименовывать, изменять цвета, начальный статус или вовсе удалять не использующиеся статусы.

<MDXImageBlock src='/images/docs/company-dm-status.png' label='Просмотр статуса.' />

# 5. Удаление

Для безвозвратного удаления статуса нажмите на соответствующую кнопку на странице просмотра записи. 
Kroncl не даст удалить статус, в котором в данный момент находятся сделки, — сначала переопределите статус сделок.

<MDXImageBlock src='/images/docs/company-dm-delete-status.png' label='Удаление статуса.' />
