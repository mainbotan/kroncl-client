# 1. Анализ сделок

Модуль управления сделками предоставляет функционал анализа сделок. Для просмотра аналитики
перейдите на вкладку <span className={styles.underline}>«Анализ»</span> раздела <span className={styles.underline}>«Сделки»</span>.

На главной странице анализа доступен просмотр основных показателей за период времени, включая:

- финансовый итог всех сделок (доходы - расходы всех зарегистрированных сделок)
- доходы
- расходы
- количество финансовых операций
- количество сделок
- количество сделок в <span className={styles.underline}>начальном статусе</span>
<br />

Кроме счётчиков на главной странице анализа доступна динамика изменения количества сделок по дням.

<MDXImageBlock src='/images/docs/company-dm-analysis-summary.png' label='Анализ сделок.' />

# 2. Период анализа

По умолчанию система показывает аналитику за <span className={styles.underline}>весь период</span> существования организации. Для изменения
периода введите целевые начальную и конечную даты, после чего нажмите на кнопку, применяющую фильтры.

# 3. По сотрудникам

Для просмотра распределения сделок по ответственным сотрудникам перейдите на вкладку <span className={styles.underline}>«По сотрудникам»</span> страницы анализа.

На странице будет показана столбчатая диаграмма, каждый столбец которой соответствует количеству сделок, за которые назначен ответственным данный сотрудник. Не забывайте, 
что каждая сделка может иметь <span className={styles.underline}>несколько</span> сотрудников.

<MDXImageBlock src='/images/docs/company-dm-analysis-employees.png' label='Распределение по сотрудникам.' />

# 4. По типам

Для просмотра распределения сделок по типам перейдите на вкладку <span className={styles.underline}>«По типам»</span> страницы анализа.

Каждому зарегистрированному <Link href='/docs/companies/modules/dm/types' className={styles.link}>типу</Link> соответствует столбец диаграммы.
<MDXImageBlock src='/images/docs/company-dm-analysis-types.png' label='Распределение по типам.' />

# 5. По клиентам

Для просмотра распределения сделок по клиентам перейдите на вкладку <span className={styles.underline}>«По клиентам»</span> страницы анализа.

Каждому зарегистрированному <Link href='/docs/companies/modules/dm/deals/client' className={styles.link}>клиенту</Link> соответствует столбец диаграммы.
