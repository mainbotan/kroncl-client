# 1. Клиент

Для привязки клиента к сделке перейдите на вкладку <span className={styles.underline}>«Клиент»</span> страницы заказа. 
Для выбора будут доступны все активные записи из контактной базы модуля <Link href='/docs/companies/modules/crm' className={styles.link}>«Клиентская база»</Link>.
