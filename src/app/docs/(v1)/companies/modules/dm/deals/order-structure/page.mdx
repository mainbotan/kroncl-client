# 1. Состав сделки

Действия просмотра и редактирования <span className={styles.underline}>состава сделки</span> доступны на вкладке 
<span className={styles.underline}>«Состав»</span> страницы заказа.

Состав сделки интегрирован с модулем <Link href='/docs/companies/modules/wm' className={styles.link}>«Каталог & Склад»</Link> и предназначен
для учёта товарных позиций (услуги/товары), проданных в процессе реализации заказа.

<MDXImageBlock src='/images/docs/company-dm-deal-positions.png' label='Состав сделки.' />

# 2. Поиск по товарным позициям

Платформа Kroncl позволяет указывать в составе заказа как абстрактные услуги и товары из <Link href='/docs/companies/modules/wm/catalog' className={styles.link}>каталога</Link>, так и товары из 
<Link href='/docs/companies/modules/wm/movement-products' className={styles.link}>поставок</Link>.

Для добавления товарной позиции начните вводить её название/идентификатор в поисковом поле. Вам будут предложены наиболее подходящие варианты
<span className={styles.link}>активных</span> товарных позиций. Чтобы добавить выбранную товарную позицию, достаточно нажатия на её строку.

# 3. Поиск товаров из поставок

Бывают сценарии, когда в сделке необходимо указать товары из конкретных поставок (партий) организации. Для поиска товаров из поставок воспользуйтесь тем же поисковым полем, что и для поиска
по каталогу, но переключите область поиска нажатием на кнопку рядом.

<MDXImageBlock src='/images/docs/company-dm-deal-positions-batches.png' label='Поиск товаров из поставок.' />

В случае добавления товара из поставки в его строке в составе сделки появится соответствующий идентификатор.

<MDXImageBlock src='/images/docs/company-dm-deal-positions-cut.png' label='Товар из поставки в составе заказа.' />

# 4. Откуда берутся цены позиций?

Цены позиций в составе заказа, вне зависимости от их источника (каталог или конкретная поставка), определяются <span className={styles.accent}>базовой ценой продажи за единицу</span>, указанной
в каталоге. Вы можете переопределить эту цену и изменить количество единиц позиции.

После изменения данных о составе заказа не забывайте <span className={styles.accent}>сохранять изменения</span> (кнопка в правом верхнем углу страницы).
