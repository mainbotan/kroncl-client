# 1. Сводка

<span className={styles.link}>«Сводка»</span> — главный экран пространства организации. Именно здесь объединены сводные данные по всем подключённым модулям и компонентам компании.

Разделение модулей и компонентов реализовано с помощью <span className={styles.underline}>виджетов</span>, каждый из которых отображает наиболее важную информацию из своей зоны ответственности. Виджеты подчиняются <span className={styles.underline}>системе разрешений</span>, благодаря чему можно легко настроить, что видит каждый сотрудник на главном экране с точностью до конкретных блоков.

Таким образом, сотрудник, не имеющий доступа к модулю финансов, не увидит соответствующий виджет на главном экране.

При нажатии на виджеты открываются соответствующие им страницы пространства организации. Здесь же находится виджет <span className={styles.underline}>«Тарификация»</span>, дающий представление об остатке дней на текущем тарифе/тестовом периоде компании.

Виджеты способны отображать не только текстовую информацию, но и диаграммы, например динамику доходов/расходов организации.
