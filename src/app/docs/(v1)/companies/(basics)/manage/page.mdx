# 1. Управление организацией

Базовое управление организацией доступно в разделе <span className={styles.underline}>«Управление»</span> (левая панель пространства организации).

Для обновления доступны данные организации, включая:

- наименование
- описание
- контактную почту представителя
- сайт организации
- регион

<MDXImageBlock src='/images/docs/company-manage.png' label='Управление организацией.' />

# 2. Красная зона

На странице управления организацией также доступны критические действия, объединённые в блок <span className={styles.underline}>«Красная зона»</span>.

<MDXImageBlock src='/images/docs/company-manage-red-zone.png' label='Красная зона.' />

# 2.1 Смена видимости

Доступны два статуса видимости организации:

- публичная (визитная карта организации доступна на основном сайте Kroncl);
- приватная (визитные данные видны только сотрудникам)
<br />
Для обновления статуса нажмите на соответствующую секцию в красной зоне и подтвердите действие.

<MDXImageBlock src='/images/docs/company-manage-update-vis.png' label='Смена видимости.' />

# 2.2 Удаление

Владельцы организации (или сотрудники, имеющие соответствующее действие максимальной критичности) могут в любой момент безвозвратно удалить <span className={styles.underline}>все данные</span> компании, включая, но не ограничиваясь: данные модулей, логи действий, настройки тарификации.

<MDXImageBlock src='/images/docs/company-manage-delete.png' label='Удаление организации.' />
