# 2. Тарифные планы

Тарифные планы градируются по количеству модулей, включённых в каждый из них. 
Не смотря на то, что все модули играют существенную роль в устройстве компании, все
их можно расположить по приоритету. 
Таким образом модуль складского учёта на платформе Kroncl имеет меньший приоритет, чем 
модуль финансов или сотрудников.

В разделе <Link href='/docs/companies/modules' className={styles.link}>«Модули»</Link> задокументированы возможности
всех модулей, здесь же перечислим состав тарифных планов.

Актаульная информация о стоимости тарифов доступна при <Link href='/platfomr/companies/new' className={styles.link}>создании организации</Link>, 
смене тарифа в настройках компании или 
<Link href='/pricing' className={styles.link}>на основном сайте Kroncl.</Link>

## 2.1 Финансист

Начальный тариф, закладывающий основу для операционного учёта организации.
Очевидно из названия, закладывает основу финансового учёта организации
(транзакционная модель модуля финансов является основной для всех других модулей).

В составе тарифа:

- <Link href='/docs/companies/modules/fm' className={styles.link}>«Финансовый модуль»</Link>: категоризация финансов, расчёт баланса организации, 
расходы/доходы, аналитика операций, долговая нагрузка;
- <Link href='/docs/companies/modules/hrm' className={styles.link}>«Управление персоналом»</Link>: карты сотрудников, должности, присвоение групповых разрешений.
